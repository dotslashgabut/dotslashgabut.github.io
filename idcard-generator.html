🖼️ Flatten (semua jadi gambar)
🧩 Vector + Layer
"Vector + Layer": teks dengan font yang bisa di-embed tetap jadi teks asli (bisa diseleksi/dicari, ukuran file lebih kecil), foto & gambar jadi objek terpisah — yang tidak didukung otomatis fallback jadi gambar per elemen (tetap sebagai layer tersendiri, bukan digabung ke background). "Flatten": seluruh kartu (background, foto, gambar, teks) dirender jadi satu gambar utuh per sisi — posisi elemen dijamin identik dengan pratinjau (paling aman terutama saat kartu dirotasi di lembar cetak), tapi teks tidak lagi bisa diseleksi/dicari dan ukuran file lebih besar.
Default nonaktif: elemen gambar .svg dijadikan gambar biasa (raster) seperti elemen lain — paling aman, karena beberapa SVG tidak tampil sempurna saat dikonversi ke vector PDF. Font tetap selalu di-embed sebagai teks asli terlepas dari opsi ini. Aktifkan kalau kamu butuh logo/ikon SVG-nya tetap tajam sebagai vector asli di PDF (tidak berlaku untuk elemen yang dirotasi/dibalik di lembar cetak, itu tetap raster).
Kalau background atau elemen gambar diunggah dari file .pdf, opsi ini menyisipkannya sebagai halaman PDF asli (vector murni, tidak dirasterisasi) menggunakan pdf-lib — hasilnya lebih rapi & tajam dibanding konversi SVG. Background hanya bisa vector kalau posisi/zoom-nya masih default (belum digeser/di-zoom); kalau tidak, otomatis fallback ke raster.
Default aktif: setiap kotak QR digambar sebagai persegi vector asli langsung di PDF (bukan gambar raster) — selalu tajam & tetap mudah di-scan di ukuran cetak berapa pun, dan hampir tidak menambah ukuran file. Otomatis fallback ke raster kalau elemen QR-nya dirotasi/dimiringkan (skew), atau kartu dirotasi/dibalik di lembar cetak.

Berguna kalau lembar hasil cetak akan dilipat di antara kartu depan & belakang: dengan kartu belakang diputar 180°, setelah dilipat kedua sisi akan sejajar menghadap arah yang benar (tidak terbalik). Hanya berlaku pada susunan atas-bawah.
