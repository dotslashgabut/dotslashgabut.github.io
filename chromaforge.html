About ChromaForge
A comprehensive color tool for designers and developers. Extract dominant colors from images, build gradients with multiple stops, generate harmonious palettes, and export in multiple formats.

Features:
  • Image color extraction with adjustable sensitivity
  • Eyedropper for precise pixel sampling
  • Linear, radial, and conic gradient builder
  • Six color harmony types based on color theory
  • Export as CSS, JSON, or PNG
  • Local library for saved palettes
